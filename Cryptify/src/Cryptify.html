FAQ

What is Cryptify?

  • Cryptify is an easy to use tool for encrypting files without installing any extra software

Is Cryptify secure?

  • Your files are encrypted within the browser and not stored on our server, in fact, your data never leaves your device

What would I use Cryptify for?

  • You can use Cryptify to make sure sensitive data is secure before sending it to someone else or just to secure data on your own computer

How do I use Cryptify?

  • Click the button, choose a file you want to encrypt and away you go - just make sure you save the encryption key or you won't be able to change your file back!

  • You should store or send your encryption key separately to the encrypted data, otherwise anyone who can see the data could potentially decrypt it

  • For example, you could e-mail an encrypted file to your friend and send them the key via text, which would make it harder for someone else to steal your data

I downloaded an encrypted file from Cryptify, how do I get the original back?

  • Just open up the html file that you downloaded after encrypting your file, pop in the encryption key you were given and your file will come back automagically

Is Cryptify open source?

  • Actually, yes ... head over to the Cryptify github page if you want to know more

Does Cryptify have any external dependencies?

  • Yes, if you enable compression Cryptify makes use of Stuart Knightley's fantastic JSZip library (under the MIT licence)
